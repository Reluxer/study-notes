# 函数中的动词选取
## 创建、销毁
create - 创建，实例创建、工厂方法
initialize - 初始化实例、属性
load - 加载配置
destroy - 销毁
uninitialize - 反初始化实例、属性

## 获取、设置
get - 通用获取属性
fetch - 获取网络请求
calculate - 计算
read - 读取文件、配置
query - 查询
find/search - 查找数据库、集合
receive - 接收文件、远程的东西
pull - 拉取
set - 通用设置属性
write - 写入文件、配置
put - 放入
push - 存入；推送通知

## 更新
reset - 重置标志、状态
refresh - 刷新页面、缓存
update - 更新配置、状态

## 添加、移除
add - 通用添加
append - 尾部追加
insert - 任意位置插入
delete - 删除
remove - 移除

## 启动、停止
open - 开启状态、打开文件
start - 开始流程
launch - 发动、启动程序
close - 关闭状态、关闭文件
stop - 停止流程
finish - 完成流程
